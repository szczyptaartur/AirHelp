in the 'examples' folder are sample data from clients from which a report is created which contains:
-Messages.txt - messages from clients
-Report.txt - statistical data from the data
-png file - a histogram of flight numbers distribution

To run the program, you must compile the AirHelp_task.py file.
To extend the program with successive file formats, in the 'run' function put another 'if' and add format_parser.py
